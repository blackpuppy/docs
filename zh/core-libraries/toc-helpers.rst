助件(*Helper*)
##############

CakePHP 包含了一些有助于建立视图的助件。助件帮助创建完整格式的标记代码(*markup*)
(包括表单)，格式化文本、时间和数字，甚至和流行的 JavaScript 库集成。这里汇总了内
置的助件。

请阅读 :doc:`/views/helpers` 来了解更多关于助件、他们的 API、以及如何创建和使用
你自己的助件。
